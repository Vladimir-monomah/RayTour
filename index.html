About Travo. - top

Travo Travel Agency & Tourism Website Script is an Online Vacantion Package & Reservation System build in PHP and MySQL. "Travo" Has individual Vacation Gallery System and Easy to Package Booking system. Our system is totaly Dynamic and from admin panel you can change A-Z of frontend. This is an complete booking system for Vacation Packages & Tours. The script is 100% OpenSource and it is easy to manage and modify the source, customization and integration in any design.

Step2: Create mySQL Database and upload SQL.sql (From Database Folder).
Step3: Goto File Manager and Copy Paste Database Information at ../core/.env File.
